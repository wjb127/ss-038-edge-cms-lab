# ss-038 edge-cms-lab — CF 무료티어 엣지 CMS 연구

**상태**: ★ 연구용(R&D). 테스트해보고 **괜찮을 때만** 실무 적용. 제품화 결정 아님.
**한 줄**: "무료·강력·소유 가능한 CMS를 Cloudflare(Workers + D1 + R2) 무료티어에 올린다"를 실측으로 검증.

---

## 동기 (왜 시작했나)
- 에이전시 랜딩·홈피에 **클라가 직접 수정하는 CMS**가 자주 필요. WP는 무료+강력해 보이지만 느림·PHP호스팅·보안 골치.
- Next+D1+R2 엣지 스택을 선호하는데 **Payload는 번들이 무거워 CF 무료티어(Worker 3MB gzip) 초과**로 보임 → 무료로 못 올림(유료 $5/mo Workers Paid 필요).
- 그래서 "WP를 AI로 뜯어 재구현?" 아이디어가 나왔으나 → **평가 결과 그건 함정**(아래).

## ★ 사전 평가 결론 (2026-06-29, 적용 전 합의)
- "WP 그대로 포팅"은 **노력 최대 + 차별화 최소**: ① Payload/SonicJS/EmDash 등 이미 점유 ② WP 가치 90%=플러그인 생태계(복제 불가) ③ 50만줄 레거시 트랜스파일=토큰지옥+평생 유지보수 부채.
- "무료"의 착시: Payload 무료초과면 답은 **$5/mo**(연 $60). WP 재구축=수백시간(에이전시 시급 환산 수백만원). $60 아끼려 그 노동 = 운영자 계산 아님.
- "AI 마이그레이션 잘함 → 재구축 가능"은 두 개를 섞은 것: **콘텐츠 이관(쉬움) ≠ CMS 엔진 재구축(어려움+부채)**.
- **정답 방향**: WP 재구축이 아니라 **CF-free에 맞게 설계된 린 CMS를 쓴다**. 자작은 기성품 다 안 맞을 때 + 학습/콘텐츠 목적일 때만, 그것도 "WP 클론"이 아니라 "필요한 10%만".

## 검증 후보 (가벼운 순 → 무거운 순)
| 후보 | 스택 | 가설 | CF무료 적합 |
|---|---|---|---|
| **★ SonicJS** | Hono + D1 + R2 + TS, edge-native | Hono가 가벼워 3MB 무료티어 실제 들어감. "Payload 무겁다"의 직접 해답 | 높음(검증대상) |
| Keystatic / Outstatic | git·md 기반, Next | 런타임 거의 0 → 무료 여유 | 높음 |
| EmDash | CF 공식 Astro, WP풍 admin | CF 네이티브 | 중~높음 |
| Payload on Workers | Next + Payload, D1+R2 자동바인딩 | 강력하나 3MB 초과 추정 → $5 유료면 해결 | 무료 X(유료면 O) |
| 린 자작 CMS | Next/Hono + D1 + R2 + admin-dashboard 스킬 | 필요한 10%만(글·미디어·어드민) = 최경량 | 최고(직접 통제) |

## 검증 기준 (실측 — 추측 금지)
1. **무료티어 적합**: 실제 배포해서 Worker gzip 번들 < 3MB 들어가나? (`wrangler deploy` 출력의 size 확인)
2. **기능**: 글/페이지·미디어(R2)·택소노미·어드민UI·권한·다국어 어디까지 기본 제공?
3. **클라 친화**: 비개발 클라가 직접 콘텐츠 수정 가능한 admin인가?
4. **한글/한국 SMB 워크플로우**: 리드폼·랜딩 구조에 얼마나 잘 맞나?
5. **DX**: AI(나)가 커스텀 잘 짜는 스택인가? (Next/TS > 올드 PHP)

## 실행 계획 (연구 단계)
1. **[1순위] SonicJS를 CF 무료티어에 실배포** → 번들 size·기능·admin 실측. `experiments/`에 결과 기록.
2. 안 맞으면 Keystatic(git기반) 테스트.
3. 둘 다 부족하면 → 린 자작 PoC(글+미디어+admin만, D1+R2).
4. 각 단계 **실측 후** 적용 여부 판단. 괜찮은 것만 실무(km 프로젝트)에 도입.

## 미결 / 주의
- WP 클론 욕심 금지 — "필요한 10%"에서 벗어나면 중단 신호.
- 무료 집착 금지 — $5/mo로 풀리는 문제면 그냥 결제가 합리적.
- 적용 전엔 어디까지나 연구. 클라 실무엔 검증 끝난 것만.
